# ADO-Monk-Chat (Front-End e Back-End + Banco de Dados) 🧑‍💻
<p align="justify">Repositório para os arquivos referentes ao front-end do projeto <strong>Monk Chat</strong>. O projeto foi construido em NodeJs, usando o Framework ReactJS, com um sistema de gerenciamento de banco de dados MySQL.</p>

## Objetivo 🎯 
<p align="justify">Este projeto full stack teve como proposito a construção de um site com crud simples baseado em um chat on-line, onde as pessoas podem ser comunicar através com outras pessoas, usando um chat simples</p>
